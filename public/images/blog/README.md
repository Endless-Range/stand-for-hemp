# Blog Images

## Style Guide for All Blog Post Images

**Consistent Illustration Style:**
- Modern editorial illustration style (think New Yorker, Politico, or The Atlantic)
- Flat design with subtle textures and gradients
- Limited color palette: hemp green (#4A7C59), earth tones (warm browns, tans), accent colors per topic
- Geometric and organic shapes combined
- Professional but artistic - not corporate or alarmist
- **NO TEXT in any images** (text will be added by the blog post template)
- Horizontal format: 1200x630px
